# echo360captions

takes the transcript and turns it into closed captioning under a video feed.

## how to use:

### 1. clone this repo

```
git clone https://github.com/t0rrential/360transcript.git
```

### 2. load it into chrome

go to `chrome://extensions/` and then enable developer mode at the top right.

next, click the `Load Unpacked` button and then select the `360transcript` folder.

### 3. enjoy!

you should be able to see transcripts on echo360 now. they will show up under a video feed after you enable it in the video player. if there is more than one video feed, after enabling closed captioning, you can click on it again to switch what video feed the caption appears under.
